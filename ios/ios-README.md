# Clean Chess iOS

Native iOS version of Clean Chess built with SwiftUI.

## Quick Start

Download the complete Xcode project: [CleanChess.zip](../CleanChess.zip)

1. Download and unzip `CleanChess.zip`
2. Double-click `CleanChess.xcodeproj`
3. Press ▶ in Xcode to run

## Files in This Directory

- `ContentView.swift` - Complete game UI and logic
- `CleanChessApp.swift` - App entry point
- `README.md` - This file
- `SETUP.md` - Detailed setup instructions

## Project Structure

The complete Xcode project (`CleanChess.zip`) contains:
```
CleanChess.xcodeproj/          # Xcode project file
CleanChess/                    # Source directory
  ├── CleanChessApp.swift      # App entry point
  ├── ContentView.swift        # Game implementation
  └── Assets.xcassets/         # Asset catalog
      ├── AppIcon.appiconset/
      └── AccentColor.colorset/
```

## Features

- ✅ Native iOS with SwiftUI
- ✅ Runs on iPhone & iPad
- ✅ iOS 15.0+
- ✅ Full chess rules
- ✅ Clean minimal UI
- ✅ New Game & Undo buttons

## Manual Setup (If Not Using ZIP)

If you want to set up the project manually:

1. Create new iOS App project in Xcode
2. Choose SwiftUI interface
3. Copy `ContentView.swift` to your project
4. Copy `CleanChessApp.swift` to your project
5. Build and run!

## Requirements

- macOS with Xcode 13.0+
- iOS 15.0+ (for running)

For detailed instructions, see [SETUP.md](SETUP.md)
